# Laboratorio-1: Familiarización con ESP32 y sensores
## Universidad de la Sabana - Internet de las cosas (2026-1)
## Jorge Esteban Diaz Bernal - Carlos Augusto Sanchez Lombana - Laura Camila Rodriguez Leon

### Objetivo General
Diseñar e implementar un sistema básico tipo osciloscopio y generador de señales utilizando los módulos ADC y DAC del ESP32, integrando elementos de interacción como pulsador y potenciómetro, además de pruebas con periféricos adicionales como MPU6050, relé y almacenamiento en microSD.

### Funcionalidades implementadas en Simulación: 
#### Osciloscopio básico
- Lectura de señal analógica por ADC.
- Visualización en Serial Plotter.
- Control de amplitud mediante potenciómetro.

#### Generador de señal
- Generación de señal analógica usando DAC.
- Ajuste de parámetros con pulsador y potenciómetro.
